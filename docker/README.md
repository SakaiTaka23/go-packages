# Docker

## インストール
- バージョンに注意すること
    - [Develop with Docker Engine API | Docker Documentation](https://docs.docker.com/engine/api/#versioned-api-and-sdk)
    - clientのバージョンがDocker本体より新しい場合は動かない
    - 必要に応じて`go get github.com/docker/docker/client`でバージョン指定する
